# MyCoffeeNote – コーヒー好み分析メモアプリ

**MyCoffeeNote** は、コーヒー豆の好みを記録・可視化・分析する、コーヒー愛好家のための専用Webアプリです。  
スマホから気軽に「購入した豆・飲んだ感想・お店の情報」などを登録でき、ChatGPTと連携した好み分析や、地図による産地の可視化機能も搭載予定です。

---

## サービス概要

- 購入した珈琲豆を登録して、品種・焙煎・精製・農園・味の感想・購入店を記録できる
- 産地を地図上に表示して、自分の好みの傾向を視覚的に理解できる
- ChatGPT連携による好み分析やおすすめ豆のレコメンド機能を予定

---

## 開発の背景（なぜ作るのか）

- 珈琲の名前や情報が個性的で覚えづらい、焙煎の情報が専門的で何で判断していいかわからない
- 個性的で覚えづらい名前のため、今までに飲んだコーヒー豆（○○○農園）が思い出せない
- お店で聞くコーヒーの味や香りの説明はピンとこないものが多いため、自分の好みにあったコーヒー豆の購入が難しい
- 豆は安くないのに、好みに合わないものを買ってしまうことがある
- 自分の好みを記録しておけば、次回購入や店員との会話がスムーズになる

---

## 想定ユーザー

### 対象
- 豆を買って自宅でコーヒーを楽しむ人
- 好みの豆・焙煎・精製・農園・お店を深掘りしたい人
- 接客の良いお店や信頼できる焙煎士を見つけたい人

### 非対象
- チェーン店のコーヒーで満足できる人
- 毎回違う豆でも気にしない人
- 記録や分析に関心がない人

---

## 利用イメージ

- スマホやPCでWebアプリにアクセス
- 登録済の豆一覧やお店情報を確認
- 自分の飲んだ豆の好み傾向を把握
- ChatGPT APIを使っておすすめの豆を提案してもらう

---

## 機能一覧

### MVP（最小実用機能）
- [ ] 豆の基本情報（品種・精製・焙煎度など）の登録
- [ ] 飲んだ感想や味の評価を記録
- [ ] 購入したお店の名前・住所を登録

### 本リリースまでに実装予定
- [ ] お店情報を地図上に表示（Google Maps）
- [ ] 産地（国・農園）の可視化
- [ ] ChatGPTと連携し、好みを分析・レコメンド
- [ ] 画像登録による簡易入力（パッケージ写真など）
